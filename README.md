# 📊 Data Analysis Projects

Projects focused on exploring datasets and practising data analysis workflows.

### 🚗 Car Sales Analysis

A small project demonstrating:

* CSV file loading with **pandas**
* structured **logging**
* safe **file handling**

The project loads a dataset and previews the data, forming a foundation for future data analysis.

---

### 💼 Job Listings Filter

A small filtering system built with **pandas** that allows users to search job postings by:

* job title
* required skills
* job location
* salary range

This project demonstrates how pandas filtering can replicate **database-style queries**.

Dataset source: [Kaggle](https://www.kaggle.com/)
