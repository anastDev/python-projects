# 🔳 QR Code Generator

A simple Python utility that generates **QR codes from text or URLs** and saves them as image files.

---

## ✨ Features

* 🔗 Generate QR codes from URLs or text
* 🖼 Export QR codes as PNG images
* 🧩 Simple reusable function

---

## ⚙️ Example Usage

```python
from qr_generator import generate_qr

generate_qr("https://github.com/your-profile")
```

Output:

```
QR Code was generated and saved to: qr_code.png
```

---

## 🛠 Technologies Used

* 🐍 Python
* 🔳 qrcode library

## 💡 Use Cases

This tool can be used to quickly generate QR codes for:
